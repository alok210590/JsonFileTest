Project Overview : The team RCB has submitted its team for upcoming match. The details of the team are as described in JSON file "rcb.json".

Tests performed :
Validate the team has 4 foreign players
Validate the team has only one wicket keeper.

What it contains:
1. Check the "rcb.json" is available in json folder path.
2. tests package contains two classes as follows:
	- JsonTest class contains methods for reading JSON file, common methods to return players data
	- TestExec class contains test cases.
	
Way to execute: Right click on TestExec.java class and run as testNG

Please check the testNG output in test-output folder by opening an index.html file OR check the console output.
